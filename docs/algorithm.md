# 算法说明

## 实验背景

迈克尔逊干涉仪对微小位移非常敏感。镜面位移或平台振动会改变两束光的光程差，摄像头中观察到的干涉条纹会随之移动。若能稳定记录条纹移动次数和方向，就可以把光学干涉现象转化为类似地震监测中的振动响应信号。

## 图像处理方法

程序每次读取一帧视频图像，先使用 CLAHE 对 LAB 色彩空间中的亮度通道做局部对比度增强。这样可以提高条纹明暗边界的可分辨性，减少环境亮度变化对判断的影响。

增强后的图像会被转换为灰度图，并做二值化处理。程序在采样框中截取局部图像，将采样区域分为左、右两部分，分别计算平均灰度值。

## 条纹计数逻辑

核心判断依据有两个：

- 左右半区平均灰度差：反映条纹在采样框中左右两侧的明暗变化。
- 采样框整体平均灰度趋势：用于判断条纹变化处于上升还是下降阶段。

当灰度差从高值越过设定阈值，同时整体平均灰度趋势发生变化时，程序认为有一条干涉条纹通过采样框。根据趋势方向不同，分别累加“从左向右”和“从右向左”的计数。

## 输出结果

程序在界面中实时显示：

- 原始视频画面与采样框位置
- 采样区域图像
- 灰度差曲线
- 平均灰度曲线
- 总计数变化曲线
- 左右方向条纹移动次数

这些结果可以用于展示外界扰动与干涉条纹变化之间的对应关系。
