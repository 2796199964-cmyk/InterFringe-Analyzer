# 使用说明

## 安装依赖

```bash
pip install -r requirements.txt
```

如果 `PySimpleGUI` 安装失败，可以单独尝试：

```bash
pip install PySimpleGUI
```

## 准备视频

将实验视频放在本地 `video/` 文件夹中，例如：

```text
video/33.mp4
```

原程序默认推荐使用 `33.mp4` 演示。GitHub 仓库不建议上传视频文件，可以在 README 中说明视频来源。

## 启动程序

```bash
python src/michelson_interferometer_monitor.py
```

启动后点击“开始”读取视频。如果视频路径不正确，可以通过界面中的文件选择按钮重新选择。

## 调整采样框

采样框应放在干涉条纹清晰、明暗变化明显的位置。采样框过大时会引入无关背景，过小时容易受噪声影响。

建议先用默认参数运行 `video/33.mp4`，观察灰度差曲线是否有明显周期变化，再微调采样框位置、大小和对比度。

## 常见问题

### 界面能打开但视频不播放

检查视频路径是否正确，或确认 OpenCV 能读取该格式的视频。

### 曲线变化不明显

尝试提高对比度参数，或把采样框移动到条纹更清晰的位置。

### 计数跳变明显

可能是采样框过小、条纹模糊或画面抖动过大。可以适当增大采样框，或者先固定摄像头视角。
