# 基于迈克尔逊干涉仪与光学摄像头的模拟地震监测算法

本项目基于迈克尔逊干涉仪产生的干涉条纹，结合光学摄像头采集到的视频图像，模拟地震监测仪对微小振动的响应过程。程序通过监测干涉条纹在采样区域内的灰度变化，判断条纹移动方向并统计条纹变化次数，从而反映外界扰动引起的光程差变化。

该项目适合用于大学物理实验、光学干涉演示、图像处理课程设计，以及“用光学方法模拟地震监测”的实验展示。

## 项目原理

迈克尔逊干涉仪中，两束相干光叠加后形成明暗相间的干涉条纹。当反射镜或实验平台受到微小振动时，两束光的光程差发生变化，干涉条纹会产生移动。

摄像头记录条纹变化后，程序在画面中选取一个采样框，提取该区域的灰度信息。通过比较采样框左右两部分的平均灰度差，以及整体平均灰度的上升或下降趋势，可以判断条纹移动方向并完成计数。

## 算法流程

1. 读取摄像头录制的视频或实验样例视频。
2. 对视频帧做对比度增强，提高干涉条纹的明暗差异。
3. 将图像转为灰度图，并根据阈值进行二值化处理。
4. 在用户设置的采样框内提取局部条纹图像。
5. 分别计算采样框左半部分和右半部分的平均灰度。
6. 根据左右灰度差和整体平均灰度趋势判断条纹移动方向。
7. 统计条纹向左、向右移动的次数，并实时显示总计数趋势。

## 环境依赖

建议使用 Python 3.9 或以上版本。

```bash
pip install -r requirements.txt
```

主要依赖：

- `opencv-python`：视频读取与图像处理
- `numpy`：灰度值计算
- `PySimpleGUI`：图形界面

## 运行方式

```bash
python src/michelson_interferometer_monitor.py
```

程序启动后，在界面中点击文件选择按钮，选择 `video/` 目录下的演示视频。推荐优先使用：

```text
video/33.mp4
```

当前项目已包含三段演示视频：

```text
video/
  31.mp4
  32.mp4
  33.mp4
```

如果只想快速演示，建议选择 `video/33.mp4`；如果想对比不同扰动效果，可以分别运行 `31.mp4`、`32.mp4` 和 `33.mp4`。

## 界面参数

| 参数 | 含义 |
| --- | --- |
| `采样框` | 调整采样区域的位置和大小 |
| `对比度` | 调整 CLAHE 对比度增强强度 |
| `基线` | 用于观察灰度变化曲线的参考阈值 |
| `从左向右（吞）` | 统计条纹向右移动次数 |
| `从右向左（吐）` | 统计条纹向左移动次数 |
| `总计数` | 两个方向计数的差值 |
